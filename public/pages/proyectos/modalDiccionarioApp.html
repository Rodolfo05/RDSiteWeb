Diccionario App
Previous Next

La temática de esta aplicación es la de un diccionario sobre conceptos de informática en general. En este proyecto desarrollé un mantenedor en donde en donde se pueden crear, modificar y eliminar "definiciones" o "palabras" de este diccionario.


Las tecnologias utilizadas en este proyecto fueron React para el front-end, el backend fue desarrollado en Node JS que a su vez se comunica con la base de datos en SQL Server la cual almacena la información del diccionario.
